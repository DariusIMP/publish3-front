# Publish3 Frontend

This repository contains the front end codebase for the [Publish3](https://github.com/DariusIMP/Publish3) project.

## Quick Start

### Prerequisites

- **Node.js** >= 20.x
- **Yarn** 1.22.22 (recommended) or npm
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/DariusIMP/publish3-front.git
   cd publish3-front
   ```

2. **Install dependencies**
   ```bash
   # Using yarn (recommended)
   yarn install
   
   # Or using npm
   npm install
   ```

3. **Set up environment variables**
   ```bash
   # Copy the example environment file
   cp .env.example .env.local
   
   # Edit .env.local with your configuration
   # Note: Check if .env.example exists, if not create .env.local manually
   ```

4. **Start the development server**
   ```bash
   # Using yarn
   yarn dev
   
   # Using npm
   npm run dev
   ```

5. **Open your browser**
   Navigate to [http://localhost:3033](http://localhost:3033)
